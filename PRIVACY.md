# WTools 隐私政策（Privacy Policy）

最后更新：2026-03-18

WTools 是一款本地运行的 macOS 工具应用。我们非常重视你的隐私。

## 1. 我们收集哪些数据？

**WTools 不收集任何用户数据。**

- 不需要账号登录
- 不包含广告 SDK
- 不上传使用数据、诊断数据或崩溃日志到我们的服务器

## 2. 数据如何被处理？

WTools 的功能（例如 Finder 右键菜单、新建文本文件、复制路径、在此处打开终端）均在你的设备本地执行。

## 3. 第三方共享

我们不会向第三方出售、共享或披露你的个人信息。

## 4. 权限说明

WTools 可能在系统层面需要你手动启用 Finder 扩展，并在你主动触发相关功能时与系统组件交互（例如打开终端）。这些操作由 macOS 系统机制执行，WTools 不会将数据发送到我们的服务器。

## 5. 联系方式

如果你对本隐私政策有任何疑问，请联系：

- Email：zhihui.jie@outlook.com
